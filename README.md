# Informační systém pro basketbalový klub „BasketIS“

## Krátký popis
Jednoduchý webový informační systém určený pro správu členů basketbalového klubu, plánování tréninků a zápasů, evidenci docházky a sledování plateb členských příspěvků.

## Cíl systému a cílová skupina

**Cílová skupina:**
- Trenéři basketbalového klubu
- Hráči (členové týmu)
- Klubový manažer nebo administrátor

**Cíl systému:**
Systém má usnadnit každodenní fungování basketbalového klubu.
Umožní přehlednou evidenci členů, tréninků a zápasů, sledování docházky a plateb.
Nahradí papírové přehledy a chatové domluvy jedním centralizovaným systémem, který zjednoduší komunikaci mezi trenéry, hráči a vedením klubu.

## Základní funkce (Role a oprávnění)

### Trenér
- Vytvářet a upravovat tréninky a zápasy (datum, čas, místo, soupeř).
- Evidovat docházku hráčů.
- Přidávat poznámky k výkonům hráčů.
- Odesílat oznámení hráčům o změnách nebo nových událostech.

### Hráč
- Prohlížet svůj profil (docházka, zápasy, statistiky).
- Potvrzovat nebo odmítat účast na tréninku či zápase.
- Přijímat zprávy a oznámení od trenéra.
- Zobrazovat klubový kalendář s akcemi.

### Administrátor / Manažer klubu
- Spravovat všechny uživatele (hráče, trenéry, týmy).
- Kontrolovat platby členských příspěvků.
- Přehledně sledovat plánované akce a obsazenost tréninků.
- Exportovat data (např. seznam hráčů nebo přehled docházky).

## Spravovaná data

- **Hráči:** jméno, datum narození, tým, pozice, kontakt, stav členství, docházka
- **Trenéři:** jméno, kontakt, týmy, naplánované akce
- **Týmy:** název, věková kategorie, trenér, seznam členů
- **Tréninky:** datum, čas, místo, účastníci, poznámky
- **Zápasy:** soupeř, místo, datum, výsledek, sestava
- **Platby:** hráč, částka, datum, stav platby (zaplaceno/neuhrazeno)
- **Oznámení:** text, datum, odesílatel, příjemci

## Možné budoucí rozšíření
- Mobilní aplikace s notifikacemi o změnách tréninků.
- Statistiky výkonů hráčů (body, asistence, fauly).
- Modul pro evidenci klubového vybavení (dresy, míče).
- Online platby členských příspěvků.

*Projekt „BasketIS“ pomáhá klubům digitalizovat jejich činnost a přináší přehled, komunikaci a efektivitu do řízení sportovního týmu.*
